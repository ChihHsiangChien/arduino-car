# Arduino 智慧遙控避障車 (Smart Car)

這是一個基於 Arduino 的智慧小車專案，結合了 **紅外線遙控 (Manual Mode)** 與 **自動避障 (Auto Mode)** 功能。使用者可以透過遙控器自由切換模式，體驗手動駕駛或讓小車自動探索環境。

## 功能特色

*   **雙模式控制**：
    *   **手動模式 (Manual)**：使用紅外線遙控器進行全方位控制 (前進、後退、轉彎、原地旋轉)。
    *   **自動模式 (Auto)**：利用超音波與紅外線感測器自動偵測障礙物並閃避。
*   **智慧避障邏輯**：
    *   前方遇障礙時自動後退並轉向尋找出口。
    *   左右側遇障礙時自動反向微調行駛方向。
*   **緊急控制**：具備緊急煞車與暫停功能，確保操作安全。
*   **蜂鳴器提示**：支援發聲功能 (Pin 11)。

## 軟體依賴

本專案使用以下 Arduino 函式庫：
*   **IRremote** (請在 Arduino IDE 的 Library Manager 中搜尋並安裝)

## 操作說明 (紅外線遙控器)

| 按鍵 | 功能 | 說明 |
| :--- | :--- | :--- |
| **EQ** | **切換模式** | 在「手動模式」與「自動模式」之間切換 |
| **PLAY/PAUSE** | **暫停 / 急停** | 自動模式下暫停/恢復；手動模式下緊急煞車 |
| **按鍵 2 / 上** | 前進 | (手動模式) |
| **按鍵 8 / 下** | 後退 | (手動模式) |
| **按鍵 4 / 左** | 左轉 | (手動模式) |
| **按鍵 6 / 右** | 右轉 | (手動模式) |
| **按鍵 1** | 原地左旋 | (手動模式) |
| **按鍵 3** | 原地右旋 | (手動模式) |

> **注意**：不同型號的遙控器編碼可能不同，若按鍵無反應，請監看 Serial Monitor 確認您遙控器的解碼數值，並修改程式碼中的 case 數值。

## 安裝與執行

1.  下載本專案代碼 (`final.ino`)。
2.  使用 Arduino IDE 開啟檔案。
3.  確認已安裝 `IRremote` 函式庫。
4.  將程式燒錄至 Arduino 開發板。
5.  開啟 Serial Monitor (Baud Rate: 115200) 查看小車狀態與除錯資訊。
